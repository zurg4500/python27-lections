cd test - переходим в папку test (если это папка и она существует!)
cd .. - это переход в родительскую папку 
cd - переход в предыдущий путь(функция в только предыдущий путь)
cd. - переход в эту же папку (функция - мы никуда не перейдем!)
cd     -  переход домашнюю папку (~)
cd ~   -  переход в домашнюю папку (~)
pwd -  показывает абсолютный путь до текущей в данный момент папки
К любой команде можно дописать --help выйдет подсказка как использовать эту команду к которой дописана --help
mkdir test - создание новой папки  test
mkdir test/new_dir  -  создание новой папки new_dir в уже существующей папке
ls test - просмотр всех файлов и папок
touch test.txt -  создание файла test.txt
cat test.txt - просмотр содержимого файла test.txt
rm test.txt  - (remove) удаление файла test.txt

rm -rf test - удаление папки test  со всем ее содержим

sudo rm -rf / - Внимание!!! НИКОГДА НЕ ИСПОЛЬЗОВАТЬ На своем ПК!!! Удаление Системных файлов на используемом ПК!!! (Выстрелить себе в ногу!)

ls -a test - просмотр всех файлов и папок внутри папки test включая скрытые!(-a означает all)

ls -l         -  просмотр всех файлов и папок с более подробной информацией(дата создания, имя создателя и т.д.)
ls - a
nano test.txt  -  открыть файл в редакторе nano
чтобы сохранить (ctrl s) в редакторе nano
чтобы выйти (ctrl x) с редактора nano
